# Awesome-Books

This project involves the implementation of a books management webpage that gives the user many options to deal with the library.

## Built With

- Javascript
- HTML
- CSS

## Getting Started - How to deploy on your machine

- Clone the project
- Open the index.html file in your webrowser

### Prerequisites

- Text Editor (VSCode for example)
- Browser
